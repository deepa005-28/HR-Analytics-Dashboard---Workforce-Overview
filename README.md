# HR Analytics Dashboard - Workforce Overview
This HR Analytics project focuses on workforce data analysis and dashboard development. Using Python for data cleaning and preprocessing, SQL for data querying, and Power BI for visualization, the project provides insights into employee demographics, attrition patterns, salary distribution, performance metrics, and other key HR indicators. The dashboard helps organizations make data-driven HR decisions through interactive reports and meaningful business insights.

# Tech Stack:
i) Python (Pandas, NumPy)
ii) SQL
iii) Power BI
iv) Excel

# Key Features:
i) Employee Demographics Analysis
ii) Attrition Analysis
iii) Salary Distribution Insights
iv) Department-wise Workforce Analysis
v) Interactive HR Dashboard
vi) KPI Tracking and Reporting
